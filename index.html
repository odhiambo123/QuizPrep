Data Structures

What is an m-ary tree

In graph theory, an m-ary tree (also known as k-ary or k-way tree) is a rooted tree in which each node has no more than m children.
A binary tree is the special case where m = 2,
A ternary tree is another case with m = 3 that limits its children to three

What is a B+ tree

A B+ tree is an m-ary tree with a variable but often large number of children per node.
A B+ tree consists of a root, internal nodes and leaves.
The root may be either a leaf or a node with two or more children.
